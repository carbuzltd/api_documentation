String helper methods

Methods

(static) positions_load(opts)

It gives position, account value and ending balance for a selected date

Example
const axios = require('axios');
axios({
  method: 'post',
  url: domain + '/api/positions_load',
  responseType: 'json',
  data: {
    sid: '[api_key]',     //The string your admin gave to you
    key: '123456789',     //Portfolio key
    toCcy: 'EUR',         //Currency in which the result will be provided
    addZeroPos: false,    //If true, it shows also closed position
    adjustFormat: true,   //If true, additional information are provided
    loadAllOrders: false, //If true, current orders will be considered
    date: '2021-01-02'    //Date for which the position is computed
  }
})
