assets

String helper methods

Methods

(static) assets_load(opts)

Get all the assets (in the database) containing the string specified in the search. If search is empty, it returns all the assets in the database.

Example
const axios = require('axios');
axios({
  method: 'post',
  url: url + 'assets_load',
  responseType: 'json',
  data: {
    sid: '[api_key]',     //The string your admin gave to you
    search: 'APPLE',      //Search for assets containing this string
  }
})
